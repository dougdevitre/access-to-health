## Usage Note

These URLs are reference starting points. When generating content:
1. Check that the link is still active before including in public materials
2. Cite the specific page or publication, not just the domain
3. Include access date for web references
4. For policy statements, include statement number and year
5. Prefer peer-reviewed sources (AJPH) over general web pages for clinical claims
